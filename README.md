 Proyecto: Plan de Respuesta a Ransomware (NIST)

Descripción
El proyecto desarrolla un plan de respuesta para la empresa ficticia TechCo, cubriendo las 5 funciones principales del NIST Cybersecurity Framework:
Identificar: Activos críticos y vulnerabilidades.
Proteger: Medidas de seguridad y prevención.
Detectar: Herramientas de monitoreo y alertas.
Responder: Protocolos de contención y comunicación.
Recuperar: Estrategias de restauración y continuidad.
